# discort-bot-open
Open source discord bot written with JS


# Setup
 - You need to have a file named botConfig.json that has your bot's token and ID.
 - And another file named youtubeConfig.json that has your youtube API key.
 
# Examples
```
{
    "token": "yourBotsToken",
    "autorun": true,
    "id": "yourBotsId"
}
```

```
{
    "key": "yourKey",
    "maxResults": 1
}
```
